US Airline sentiment on Twitter

Twitter data taken from Kaggle, processed, and visualised using d3.js

These donut charts display the share of each airlines sentiment on Twitter, that is the positive, negative and neutral tweets about the airline. Data taken from CSV files and chart created using D3.js, making use of the d3.pie() functionality.

Select an airline from the dropdown menu below to see the percentage of positive, negative, and neutral tweets.
